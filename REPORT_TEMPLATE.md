# Mini Project Report Template

Use this template to write your final PDF report required by the assignment. Include screenshots for every step.

Sections to include:
- Title page (Roll number, Name)
- Problem Statement
- What and Why of DevOps
- Tools used (brief)
- Step-by-step (Source control -> Testing -> Build -> CI -> Docker -> Docker Hub -> Ansible)
- Commands used (copy paste)
- Screenshots (GitHub repo, Jenkins pipeline, Docker Hub repo, Ansible output, Application output)
- Links (GitHub repo URL, Docker Hub URL)
- Scripts appended (Dockerfile, Jenkinsfile, playbook)
